# EastSide Dark — дизайн-система

Эталон стиля — страница анкеты **`anketa-dark.html`** («Где ты сейчас?»).
Этот документ описывает ее язык до мелочей, чтобы любую новую страницу
(человеком или нейронкой) можно было собрать неотличимо от нее.
Страницы в этом стиле: `anketa-dark.html`, `diagnostic-dark.html` («Разбор»),
`loading-dark.html` (экран генерации), `roadmap-dark.html` («Твое восхождение» —
кинематографичная фото-гора с прожигом тропы, HUD целиком в этом языке:
Arkhip/Onest, сапфир, внутренние свечения; открывается из разбора большим
попапом, умеет embed-режим через postMessage `roadmap-close`).
Старый прототип `prototypes/v2/ignite-demo.html` (Unbounded/Golos) — архив,
с него снят движок; новые правки — только в `roadmap-dark.html`.

---

## 1. Суть стиля в трех предложениях

Ночное небо глубокого синего, на котором лежит светлая молочная плашка с
контентом, а на ней — тёмная панель-спутник. Премиальный, спокойный,
«дорогой» вид: мало текста, крупные значения, тонкие белые обводки и
фирменное **свечение изнутри** на всем выбранном и активном.
Метафора бренда — восхождение: векторная гора со светящейся тропой из точек
и звездой на вершине живет в тёмной панели каждой страницы.

## 2. Композиция — трёхслойка (load-bearing)

Каждая страница платформы собрана из трех слоев. Это главный, узнаваемый
прием — без него страница не наша.

```
┌─ СЛОЙ 1: тёмная база #03081C ─────────────────────────────────┐
│ ┌─ сайдбар ─┐  ┌─ СЛОЙ 2: светлая плашка ──────────┬────────┐ │
│ │ 240px     │  │  контент в колонке max 824px      │ СЛОЙ 3 │ │
│ │ тёмный,   │  │  (центрирована)                   │ тёмная │ │
│ │ без фона- │  │                                   │ панель │ │
│ │ карточки  │  │                                   │ 312px  │ │
│ └───────────┘  └───────────────────────────────────┴────────┘ │
└────────────────────────────────────────────────────────────────┘
```

Точная геометрия:

- `body` — `overflow: hidden`, скроллится только внутренность плашки.
- `.shell` — `display: grid; grid-template-columns: 240px 1fr; gap: 16px; height: 100vh; padding: 16px;`
- Светлая плашка `.plate` — `border-radius: 24px; overflow: hidden;` занимает всю правую ячейку.
- Тёмная панель `.path` — НЕ третья колонка грида, а `position: absolute; top: 16px; right: 16px; bottom: 16px; width: 312px;` — она **лежит на плашке** с зазором 16px со всех сторон.
- Скролл-зона плашки `.plate-scroll` — `padding: 64px 56px 22px; margin-right: 344px` (344 = 312 панель + 16 зазор + 16 воздух). Скроллбар скрыт (`::-webkit-scrollbar { width: 0 }`).
- Колонка контента `.plate-col` — `max-width: 824px; margin: 0 auto;` — контент не растекается по ширине.

Фон базы — не плоский черный, а с двумя тихими синими переливами:

```css
body { background: #03081C; }
body::before {  /* fixed, pointer-events: none */
  background:
    radial-gradient(900px 500px at 85% -10%, rgba(30,114,240,.16), transparent 60%),
    radial-gradient(700px 500px at 0% 110%, rgba(22,100,230,.12), transparent 60%);
}
```

## 4. Типографика

Два шрифта, всегда одни и те же:

- **Arkhip** — фирменный, ТОЛЬКО заголовки. Локальный файл
  `assets/fonts/arkhip/Arkhip_font.ttf` (+.otf), `font-weight: 400` (других нет).
  Фолбэк-цепочка: `'Arkhip', 'Poiret One', 'Onest', sans-serif`.
- **Onest** (Google Fonts, 400–900) — все остальное: текст, кнопки, цифры, лейблы.

Шкала (px):

| Роль | Размер | Вес | Межбуквенное | Шрифт |
|---|---|---|---|---|
| H1 страницы | 40 | 400 | −3px (короткий) / −2.4px (длинный) | Arkhip |
| Заголовок панели | 25 | 400 | −0.8px | Arkhip |
| Имя в профиле | 17 | 400 | +0.8px | Arkhip |
| Большой балл (78) | 56 | 800 | −0.05em | Onest |
| Крупное значение в карточке | 22 | 800 | −0.025em | Onest |
| Лейбл вопроса (q-label) | 16.5–17 | 700 | — | Onest |
| Заголовок карточки-опции | 15–16 | 700 | — | Onest |
| Пункт навигации | 14.5 | 500 (актив 600) | — | Onest |
| Подзаголовок (sub) | 15 | 400, line-height 1.6 | — | Onest |
| Подсказка (q-hint) | 13.5 | 400 | — | Onest |
| Вторичная строка карточки | 13 | 400, line-height 1.42 | — | Onest |
| Мелкий метатекст | 12–12.5 | 500–600 | — | Onest |
| Капс-лейблы | 10–12 | 700–800 | +.04em…+.08em, uppercase | Onest |

Правила:

- Цифры всегда `font-variant-numeric: tabular-nums`.
- **Максимум два уровня текста на карточку**: заголовок + одна строка. Третий
  уровень — только если это капс-лейбл или значение, не предложение.
- Заголовки никогда не светятся и не красятся в акцент — цвет несут элементы.
- На «/100», «/год» и прочие приставки — уменьшенный кег (0.3em от числа), цвет muted.

## 5. Свет — фирменный прием

Главное правило: **на поверхностях свечение всегда ВНУТРИ (inset), не вокруг**.
Внешний ореол (`box-shadow` без inset) на карточках и кнопках — запрещен.
Наружное свечение разрешено только мелким светящимся элементам: точки, бары,
иконки, кольца — через `box-shadow 0 0 8–14px` или `filter: drop-shadow`.

Эталонные формулы (копировать, не изобретать):

```css
/* выбранная карточка на светлом (чойс-чипс, «твоя ставка», итог-строка) */
.sel {
  background: rgba(255,255,255,.9);
  border: 1.5px solid rgba(43,143,255,.4);
  box-shadow: inset 0 0 26px rgba(43,143,255,.3), inset 0 0 6px rgba(43,143,255,.16);
}

/* главная кнопка */
.btn-next {
  background: #2073E6; color: #fff;
  box-shadow: inset 0 0 18px rgba(120,190,255,.85), inset 0 1px 0 rgba(255,255,255,.3);
}
.btn-next:hover { background: #2B8FFF; }

/* светящийся чип-иконка (28×28, radius 9) */
.chip-glow {
  background: rgba(43,143,255,.14); color: #EAF2FF;
  box-shadow: inset 0 0 14px rgba(43,143,255,.95), inset 0 0 3px rgba(160,205,255,.7);
}
/* красная версия (выход/опасное): те же значения, цвет rgba(255,85,85,…) */

/* активный пункт навигации: градиент слева направо + внутренний свет */
.nav-active {
  background: linear-gradient(90deg, rgba(43,143,255,.55) 0%, rgba(43,143,255,.22) 45%, rgba(43,143,255,.06) 100%);
  border: 1px solid rgba(60,150,250,.7);
  box-shadow: inset 0 0 22px rgba(43,143,255,.28), inset 0 1px 0 rgba(255,255,255,.16);
}
/* иконка активного пункта светится сама: */
.nav-active svg { filter: drop-shadow(0 0 8px rgba(185,215,255,1)) drop-shadow(0 0 18px rgba(43,143,255,1)) drop-shadow(0 0 4px #fff); }

/* активная строка на тёмной панели */
.step-active {
  background: linear-gradient(120deg, #133A6E, #0D2C62 75%);
  border: 1px solid rgba(60,150,250,.45);
  box-shadow: inset 0 0 18px rgba(30,114,240,.22), inset 0 1px 0 rgba(255,255,255,.1);
}

/* тёмная карточка-виджет (компас, навигатор) */
.dark-widget {
  background: #07153C; border: 1px solid rgba(43,143,255,.28);
  box-shadow: inset 0 0 26px rgba(43,143,255,.22), inset 0 1px 0 rgba(255,255,255,.06);
}
```

## 6. Материалы поверхностей (светлая плашка)

Иерархия — ровно три материала, всегда одинаковые:

1. **Контейнер раздела `.q-card`** — почти невидимая вуаль:
   `background: rgba(255,255,255,.28); border: 1px solid rgba(255,255,255,.7); border-radius: 22px; padding: 24px 26px;`
   Внутри: `q-label` (16.5–17/700, `#21212E`) + `q-hint` (13.5, `--ink-sub`) + сетка.
2. **Карточка-опция `.opt`** — матовое стекло:
   `background: rgba(255,255,255,.42); border: 2px solid rgba(255,255,255,.96); border-radius: 16px; padding: 15–17px 17–19px;`
   Hover: `background: rgba(255,255,255,.6); border-color: rgba(43,143,255,.55);`
3. **Выбранное/главное `.sel`** — формула из раздела 5.

Никакого `backdrop-filter` на карточках (дорого и не нужно — вуаль дает эффект).
Исключение — футер-бар: `rgba(229,232,251,.8)` + `backdrop-filter: blur(12px)`.

Сетки: `gap: 14px`, колонки по контенту (2 или 3). Разделители внутри карточки —
только хайрлайны `1px solid rgba(36,40,90,.12)`.

Порядок сверху вниз: кикер (12.5, muted) → заголовок Arkhip 25 →

  Внутри — SVG-гора (раздел 8).
- **Прогресс-бар**: трек `#07112E` h6 radius 99; филл — градиент
  `90deg, #9FCBFF → #5CB4FF` + `box-shadow: 0 0 10px rgba(43,143,255,.8)` +
  бегущий блик (`::after` 45% ширины, белый градиент, анимация 2.4s).
- **Строки-факты `.step`**: `padding: 9px 12px; radius 13;` кружок 25–27px с

### 7.4 Экран загрузки (`loading-dark.html`)

Ободран до кости: фон базы, в центре ОДНА крупная фраза (Onest 800,
clamp 28–52px), слова всплывают из своих масок с блюром
(`translateY(118%) + blur(7px) → 0`, 1s `cubic-bezier(.16,1,.3,1)`, задержка
`i × 55ms`); уход — вверх 0.44s. Снизу — числовой счетчик (Space Grotesk,
монотонно вверх) и строка-подсказка. Сверху — капс-кикер с пульс-точкой.
Больше НИЧЕГО: ни горы, ни карточек, ни логотипа-плашки.

## 8. Векторная гора

Правило: гора, тропа, точки и звезда — **всегда один SVG в одной системе
координат** (`viewBox="0 0 280 142" preserveAspectRatio="none"`). Никаких
растровых гор с наложенными поверх точками — разъезжается.

Состав (порядок слоев):

1. Радиальное свечение за вершиной (`ellipse` cx 212 cy 40).
2. Фоновые звезды — 6–8 кружков r 0.7–1.1 `#CDD2FF` opacity .35–.55.
3. **Low-poly склон**: гребень идет по точкам
   `(0,132) (34,126) (66,120) (96,112) (124,98) (150,84) (176,62) (196,46) (212,30-вершина)`,
   спуск `(232,54) (252,80) (272,104) (280,112)`. Каждый сегмент — две грани
   (треугольника): левый склон светлее (`#1A3A9C…#2A52C2`), правый в тени
   (`#102E64…#071846`). Снежная шапка у вершины `#79A6F2`.
4. Передний тёмный хребет для глубины (`#0E2A60`, `#0A1E50`, opacity .9).
5. Тонкие ребра граней `#091540` stroke .7 opacity .5.
6. Светящийся гребень: polyline по точкам гребня,
   `stroke: rgba(120,185,255,.7)` 1.4 + фильтр `glowDot`
   (`feGaussianBlur stdDeviation 2.2` + merge с исходником).
7. Тропа: пунктир `stroke-dasharray: 1 5` `rgba(120,185,255,.5)`.
8. **7 точек маршрута** на гребне, r 3.1–3.8, цвета от `#3FA2FF` (низ) к
   `#B0DCFF` (верх), все под фильтром `glowDot`. Прогресс показывается
   прозрачностью точек (непройденные — opacity ~.3).
9. Звезда на вершине (212,28): ромб-четырехлучевик `#EAF4FF` + радиальный
   ореол r19 + тонкие лучи-кресты. Если цель не достигнута — вся группа opacity .55–.6.

## 9. Движение

Разрешенный набор (все — однократные на появление или тихие циклы):

| Что | Параметры |
|---|---|
| Заполнение баров/колец | `transition .9–1.2s cubic-bezier(.22,.61,.36,1)`, delay .25–.3s, из нуля |
| Счетчик большого числа | rAF, ease-out-cubic, ~1.1s |
| Бегущий блик по филлу | 2.4s ease-in-out infinite (см. 7.3) |
| Пульс точки-статуса | scale/opacity 1.4–1.7s infinite |
| Каскад мелких элементов | задержка 80–90ms на элемент |
| Hover карточек | только background/border/box-shadow .15s; на тёмном допустим translateY(-2px) |
| Смена шага/контента | fade+slide 8–18px, 150–250ms |

Запрещено: parallax, scroll-jacking, fade-in-on-scroll на каждом блоке,
вращения, scale-bounce на карточках, анимации дольше 1.2s вне загрузки.

## 10. Иконография

- Только line-иконки: `stroke: currentColor; stroke-width: 1.8–1.9` (мелкие 2–2.6),
  `stroke-linecap/linejoin: round`, без заливки (заливка только у точек-кружков).
- Размеры: 19px навигация, 14–17px внутри чипов и строк.
- Рисуются инлайном в SVG 24×24, не иконочными шрифтами.
- Эмодзи как иконки — запрещены.

## 11. Текст и тон

- Голос `eastside-copy`: на «ты», живой, без канцелярита и маркетингового пафоса.
- **Без буквы «ё»** везде (кроме имен собственных).
- Числа: разряды через неразрывный пробел (`240&nbsp;000 ₽`), валюта после числа.
- Никаких «гарантируем 100%», таймеров и давления — честность это стиль
  (барьер аудитории — недоверие; см. product-research).
- Заголовки персональные, когда есть имя: «Шансы хорошие, Артем».
- Лейблы статусов — одним словом: «сильно / средне / слабо», «твоя ставка»,
  «цель-максимум», «с запасом».

## 12. Запреты (нейрослоп-лист)

Если поймал себя на любом из пунктов — остановись и убери:

1. Абзац мелкого текста под каждым элементом сетки. Карточка = заголовок + одна строка.
2. Коробка в коробке в коробке (контейнер → лига → карточка → подкарточка). Максимум два уровня вложенности.
3. Нумерованные кикеры секций («01 · РАЗБОР») и прочая декоративная шелуха.
4. Радары, спидометры, донаты-диаграммы. Уровень показывают бары и кольца.
5. Внешние ореолы у карточек и кнопок (свечение — внутрь, см. раздел 5).
6. Фиолетовый, пурпурный, «AI-градиент» фиолетово-розовый. Акцент один — сапфир.
7. Чистый `#000`/`#fff` как фон. У нас `#03081C` и молочная плашка.
8. Glassmorphism с настоящим blur на каждой карточке (только футер-бар).
9. Эмодзи, стоковые 3D-иллюстрации, дженерик-иконки с заливкой.
10. Буква «ё», канцелярит, «Вы» с большой буквы.
11. Серые рамки `#ccc`/`#ddd` — обводки у нас белые полупрозрачные или синие.
12. Новые шрифты. Arkhip + Onest, точка.
